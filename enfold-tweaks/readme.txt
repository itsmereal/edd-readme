Customization and Improvements for the Enfold Theme.

== Description ==

Supercharge your Enfold theme with customizations and improvements. Enfold Tweaks plugin comes with plenty of features including:

* Clean up WordPress Admin Bar
* Extend Avia Layout Builder for custom post types.
* Post excerpt length customization
* Modern header banner design
* Codemirror support for the code block element
* Allow SVG Upload
* Custom admin bar links
* Additional recovery email for white screen of death situations
* Extra Google fonts support
* Extra image thumbnail sizes
* Quick search for the enfold icon font
* Extra Avia Layout Builder Widgets
* And many more...

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Navigate to "Theme Options > Enfold Tweaks" or click "Enfold Tweaks" on the admin bar to display plugin settings.

== Frequently Asked Questions ==

= Is this only for Enfold theme? =

Yes, Enfold Tweaks plugin only works with the Enfold Multipurpose theme by Kriesi.
